# AI_model_classes
This is an idea to specify loose levels / classes for AI models.

Sice almost half a Year I had that Idea and as often as I tried calculations to find another class-model, I still came back to his.
The Formular is `7^x*10-9`, where x is the class / level and the result is the first Parameter count of the class / level in Billion.


Level 01 7^1 `7` *10e-9 = `0,000000007 B`

Level 02 7^2 `49` *10e-9 = `0,000000049 B`

Level 03 7^3 `343` *10e-9 = `0,000000343 B`

Level 04 7^4 `2401` *10e-9 = `0,000002401 B`

Level 05 7^5 `16807` *10e-9 = `0,000016807 B`

Level 06 7^6 `117649` *10e-9 = `0,000117649 B`

Level 07 7^7 `823543` *10e-9 = `0,000823543 B`

Level 08 7^8 `5764801` *10e-9 = `0,005764801 B`

Level 09 7^9 `40353607` *10e-9 = `0,040353607 B`

Level 10 7^10 `282475249` *10e-9 = `0,282475249 B`

Level 11 7^11 `1977326743` *10e-9 = `1,977326743 B` # 2B to 13B - Small AI models / AI Companion Class. 

Level 12 7^12 `13841287201` *10e-9 = `13,841287201 B` # 13B to 96B - Human level AI / Expert Class.

Level 13 7^13 `96889010407` *10e-9 = `96,889010407 B` # 96B to 678B - Superhuman level AI / AGI Class.

Level 14 7^14 `678223072849` *10e-9 = `678,223072849 B` # 678B to 4,7T ASI Class.

Level 15 7^15 `4747561509943` *10e-9 = `4747,561509943 B`
